INFERÊNCIA | CLASSIFICAÇÃO

Para funcionar corretamente, é muito importante que todos os vídeos tenham sido capturados pela mesma câmera e tenham as mesmas dimensões.

PREDIÇÃO

Filtro:
Este número define o a quantidade de classificações seguidas que devem ser feitas antes de considerar o valor como válido. Utilize este filtro para evitar falsos positivos. Lembrando que valores maiores farão com que demore para reconhecer as mudanças de predição.
Registros Salvamento:
Este numero define quantos registro serão acumulados antes de salvar um arquivo zip com um log das predições e imagens. Vale o que acontecer primeiro, a quantidade de registros ou a diferença de timestamp.
Timestamp Salvamento:
Este numero define a cada quantos minutos a diferença do timestamp dos registros serão comparados antes de salvar um arquivo zip com um log das predições e imagens. Vale o que acontecer primeiro, a quantidade de registros ou a diferença de timestamp.
Nome do Vídeo:
Nome que será usado pra salvar o arquivo de Vídeo
Tempo de Captura (segundos):
Tempo de duração da gravação. Após esse tempo a gravação para automaticamente.
